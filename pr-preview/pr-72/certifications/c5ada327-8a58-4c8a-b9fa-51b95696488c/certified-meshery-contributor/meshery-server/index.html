Meshery Server Contributor Exam

The Meshery Server examination verifies contributor understanding of one of Meshery’s core architectural components and is one of a set of mandatory exams comprising the Certified Meshery Contributor certification.

This exam focuses on the candidate’s ability to contribute to the Meshery Server codebase. The exam covers a variety of topics, including:

  • Contributing to Meshery Server
  • Contributing to Meshery Server Events
  • How to write MeshKit Compatible Errors
  • Build & Release (CI)
  • Schema-driven development in Meshery

This exam does not cover Meshery Server usage, but focuses on Meshery Server contribution. For information on using Meshery Server, please refer to the Meshery Documentation.
